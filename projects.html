Experimento de IoT

  • python
  • arduino
  • IoT

Instalação em ambiente arduino de miniestação climática desenvolvido como parte do ITI MBA. Usei um microcontrolador ESP8266, um adaptador USB e um sensor de temperatura e umidade DHT11 e precisei fazer um botão para ligar o ESP em modo programação e uma fonte 5V para o sensor. Os dados de temperatura e umidade foram automaticamente alimentados em um canal no Thingspeak conforme o código em python montado no Google Collab. Ao término do experimento, pude fazer um dashboard e tirar conclusões a partir dos dados coletados.
